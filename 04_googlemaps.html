My Google Map


WHY I BUILT THIS (December 02, 2017)

Because adding a google map to your website is cool.

This wasn't the first time that I worked with the google maps API but when I did use it I really didn't know what I was doing. As the time, I just followed the documentation to the "T" and it worked.

Now, I went back to understand it. And to my surprise it wasn't as bad as I though it would be.

Zoom In and you will see the marker is placed in Berlin, Germany. This was were I learned how to code and also where I was excepted to add a googles map to my group project. Back then it was a very stressful time...looking back at it now I don't see what the big deal was.

I like these maps...I definitely want to do more projects with them...

This is just the beginning....
